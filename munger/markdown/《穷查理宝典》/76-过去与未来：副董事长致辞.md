# 过去与未来：副董事长致辞

查理·芒格对伯克希尔50年的评论

文/查理·芒格  
译/刘宁  
伯克希尔·哈撒韦《2014致股东信》第39页，2015年2月28日

致伯克希尔·哈撒韦股东：

50年来，我亲眼见证了伯克希尔在沃伦·巴菲特的领导下取得的非凡成就。在他的贺词之后，我也想从以下五个方面做些补充，给大家提供一些我的看法：

1.阐述是什么样的管理体系和政策，让昔日濒临破产的纺织企业华丽转身成为今日强大的伯克希尔；

2. 解释这一管理体系和政策的发展历程；

3. 对伯克希尔为什么如此成功作点说明；

4. 预测一下如果巴菲特在不远的将来卸任的话，伯克希尔的非凡业绩是否能够延续；

5. 探讨伯克希尔过去五十载的辉煌成就是否具有借鉴意义。

公司创建之初，巴菲特就制定了伯克希尔管理体系和政策（本文统称为“伯克希尔体系”）。这一体系具有以下特征：

1.伯克希尔可以是一家多元化的集团公司，但不会涉足无法有效预测的业务。

2. 集团公司的绝大多数业务可以由旗下独立的子公司经营，各子公司的首席执行官（CEO）可以拥有极大的自主经营权。

3. 集团公司总部可以仅有一个很简单的办公室，只包括董事长、首席财务官和几名助理。助理主要负责协助首席财务官做审计、内控等事务。

4. 伯克希尔的下属子公司一定要包含财产保险公司。这些保险公司在适当的时候，一方面为集团带来预期稳定的承保收益，另一方面为投资提供可观的“浮存金”（即未付保险负债）。

5. 集团公司不会设置统一的人事、股票期权、退休制度或相类似的其他激励机制，因为子公司会设立各自独立不同的体系和制度。

6. 仅有以下少数事务会由伯克希尔的董事长亲自掌管。

（1）负责伯克希尔几乎全部的证券投资，这些投资通常通过旗下的保险公司开展。

（2）决定所有重要子公司CEO的人选及薪酬，并请每位CEO私下推荐一名接班人，以备不时之需。

（3）子公司提升竞争优势后所产生的多余现金由他（巴菲特）负责部署投资。理想的部署是用现金去收购新的子公司。

（4）对任何子公司CEO的见面请求，基本做到有求必应；但几乎不会主动约见。

（5）每年从一个被动股东的角度出发，撰写一封篇幅较长、逻辑缜密、实用的致股东信，放在年报里同时发布；并在股东大会上花上几个小时亲自解答股东的提问。

（6）无论在任上还是任下，要在长期有利于客户、股东及其他利益相关者的企业文化中，努力成为典范，且这一文化不受其任期影响。

（7）首要任务是抽出较多的时间，投入到安静的阅读及思考中，尤其是在必学的领域，做到活到老学到老。

（8）还要投入大量的时间，饱含热情地赏识他人所取得的成就。

7. 收购新的子公司时，通常采用现金而不是发行新股的形式。

8. 只要单位留存收益为股东创造的价值比市场价值高，伯克希尔就不会发放股息。

9. 收购新的子公司时，伯克希尔会寻求董事长完全看得懂的好生意，支付公允的价格。同时，伯克希尔还希望有能力胜任的CEO在位，无须依靠集团总部的支持，能够长期有效地管理好该子公司。

10. 伯克希尔选择子公司的CEO要确保人选是可靠、能力卓越、精力充沛的，且热爱自己所处的环境和行业。

11. 伯克希尔遵循的首要准则是，几乎永远不会出售其子公司。

12. 伯克希尔几乎从不会在无关联的子公司间调动CEO。

13. 伯克希尔从不规定子公司CEO的强制退休年龄。

14. 伯克希尔几乎没有未清偿账务，因为它努力保有：（1）在任何环境下，近乎完美的信用；（2）遇到千载难逢的机会时，可以随时取得现金和信用贷款。

15. 伯克希尔永远是目标大型企业的友好买方。这类企业的出售提议会快速得到关注。如果不成交，那么只有伯克希尔的董事长及其他一、两个人知道内情，他们也绝不会向外人披露。

伯克希尔管理体系的要素和规模是绝无仅有的。我所了解的大型企业，其体系所具备的要素没有一家能达到伯克希尔的一半。

伯克希尔为什么会有如此与众不同的企业性格？

巴菲特年仅34岁时，就已持有伯克希尔45%左右的股份，赢得所有大股东的完全信任。他可以随心所欲地制定任何体系；伯克希尔体系就此诞生。

巴菲特笃信他精挑细选的每个要素都能有助于他引领伯克希尔创造辉煌成就。但他没有试图设计一套放诸四海皆准的体系。事实上，伯克希尔不要求旗下子公司采用该体系；而且一些推行不同体系的子公司，也发展得红红火火。

巴菲特设计伯克希尔体系的初衷是什么？

多年来，我观察总结了几个要点：

1.他特别希望体系中最重要的人物，不断追求在理性、能力、投入与奉献方面做到极致；并从他自己做起。

2. 他希望处处取得双赢，例如待人忠诚收获忠诚。

3. 他希望决策能由长期坚守、担当后果的人员制定，实现长期业绩最大化。

4. 他希望总部摆脱官僚制度，最大程度降低其必然导致的不良影响。

5. 他希望跟随导师本杰明·格雷厄姆的脚步，身体力行地传播悟到的智慧。

巴菲特建立伯克希尔体系之初，是否已预见之后产生的效益？答案是否定的。有些效益是在实践摸索中偶然发现的。可以肯定的是，在学习过程中，他不断地强化行之有效的做法。

巴菲特掌管下的伯克希尔业绩辉煌的原因何在？

我只想到了四大因素：

1.巴菲特本人富有建设性的特质；

2. 伯克希尔体系富有建设性的特质；

3. 好运气；

4. 来自一些股东和崇拜者（包含媒体在内）极其强烈、极具感染力的投入与奉献。

我相信这四大因素都存在，都发挥了作用。但是主要的推动力源于人与体系的特质，追崇者的投入与奉献，以及三者之间的相互作用。

特别值得指出的是，巴菲特将注意力高度聚焦于少数（投资）活动，并坚守了50年之久，产生了巨大的lollapalooza效应。巴菲特与网球名将费德勒获得成功的原因相同，就是坚持不懈。

其实，巴菲特采用了篮球名帅约翰·伍登的制胜法。伍登知道，要赢球就尽量让最优秀的七名球员打满全场；他坚持这一战术，屡试不爽。如此一来，对手永远面对第一而非第二梯队的球员。此外，有了更多上场时间，最优秀的球员也比一般球员进步更大。

巴菲特将伍登制胜法运用得更加出神入化。他把七人的技能训练集于一身，在过去50年里随着年龄的增长不断进步，而不是像篮球运动员一样江河日下。

巴菲特还在CEO层面创造强有力的伍登效应。他将权力和权威集中交给屈指可数的CEO，让他们长期掌管重要的子公司。伍登效应有助于CEO素质能力的提升，和子公司业绩的提高。

伯克希尔体系赋予许多子公司及其CEO所渴望的自主权，业绩越做越好，名声越来越大。自然而然的，伯克希尔吸引了更多更好的子公司和CEO加入。

优质的子公司和称职的CEO不太需要总部的关注，从而形成了所谓的“良性循环”。

伯克希尔坚持将财产保险公司作为重要业务的战略是否成功？

难以置信的成功！伯克希尔的鸿鹄之志似乎难以实现，但的确成功了。

通常财产保险公司投资在普通股票上的价值与其股东权益价值相当，伯克希尔的保险子公司也不例外。标普500指数在过去50年里，实现10%左右的税前年收益率，财产保险公司也搭上了这部“顺风车”。

巴菲特时代的前数十年间，伯克希尔旗下保险公司持有的普通股的表现远远好于标普500指数。这正符合巴菲特本人的预期。之后，由于伯克希尔持有股权的规模和所得税的因素，高于指数的收益部分渐渐回归指数（或许不是永远回归）。恰逢其时，更好的优势出现了。阿吉特·杰恩从零开始创造了一家强大的再保险公司，为集团带来可观的“浮存金”，同时还产生巨额的承保收益。美国政府雇员保险公司（GEICO）的业务悉数纳入伯克希尔集团，之后其市场份额增加了四倍。集团旗下的保险业务得到巨大的提升，主要得益于良好的声誉、严谨的承保政策、利基市场的发掘和占据，以及杰出人才的吸引和留存。

再之后，伯克希尔独一无二且相当可靠的企业特质及庞大规模变得广为人知。旗下的保险公司获得并抓住许多独家的、诱人的机会，购入私募证券。这类证券大多都有固定的期限与出色的表现（回报）。

伯克希尔在保险业取得骄人成绩不是自然的结果。一般而论，就算是管理出色的保险企业，也只有平平的业绩。这样的业绩对集团的贡献甚微。但是伯克希尔的业绩确是如此惊人。我甚至认为，假设回到起点，单凭睿智和青春，巴菲特自己也无法复制。

伯克希尔的多元化战略是否有不良影响？不但没有，集团的机会还因为业务的广度得以拓展。另外，巴菲特有能力化解该战略常见的负面影响。

伯克希尔为何偏向使用现金而不是自己的股票收购企业？道理简单，用伯克希尔的股票换来的企业，至少要具有同等价值；符合条件的企业实属凤毛麟角。

一般而言，收购案对收购方的股东不利。为什么伯克希尔非保险业的收购却相当成功地为股东创造了价值，而不是摧毁价值呢？

这要归功于伯克希尔体系的设计，拥有方法论上的优势得以把握更好的机会。伯克希尔没有所谓的“收购部”，也就没有这类部门面对的业绩压力。它也从不依赖所谓的“撮合者”的意见，也就不会被成交偏见所左右。此外，巴菲特自知且内敛。作为长期的被动投资人，他积累了相当丰富的经验，对于企业经营的情况比大多数的高管更在行，尽管他并不显露自己的专业能力。最后，尽管伯克希尔掌握着大多数企业无法匹敌的好机会，巴菲特依然表现出超乎常人的耐心，出手次数少之又少。例如，在担任伯克希尔掌门人的前十年，巴菲特经历了一家企业（纺织厂）的没落，两家新企业的收购；加减之后，净添一员。

伯克希尔在巴菲特掌管期间出现过哪些重大失误？虽然“作为”导致的错误更常见，但是几乎所有的重大失误都属于“不作为”，即不买入。这类错误包括尽管确信沃尔玛股票有巨大潜力，却未能出手买入。不作为的后果相当严重。伯克希尔缺乏足够的智慧，没有看准几乎是十拿九稳的好机会，错过了几个良机；代价至少是500多亿美元的净值。

现在到了倒数第二个问题：假设巴菲特即将卸任，伯克希尔的非凡业绩是否能够延续。

答案是肯定的。伯克希尔的子公司具有基于持久竞争优势的发展动力。此外，旗下铁路和公共事业子公司提供大型固定资产投资的理想机会。许多子公司正在进行明智的“补强”收购。

在目前发展势头和投资机会并存的大好形势下，即使假设：（1）巴菲特明天卸任，（2）接班人是平庸之辈，（3）伯克希尔永远不再收购大企业，只要伯克希尔体系基本保持不变，在未来相当长的时期内，保持超常的业绩几乎是毫无疑问的。

然而事实是，如果巴菲特即将卸任，他的接班人可不是什么“平庸之辈”。我只说说两位：阿吉特·杰恩、格雷格·阿贝尔。两人的能力久经考验，业绩有目共睹；往小了说，实属“世界一流”，让我说，堪称“世界顶尖”。在一些重要的方面，与巴菲特相比，每位都是更出色的高管。

此外，我相信任何一位都不会：（1）为别人的天价高薪所动，离开伯克希尔；（2）希望伯克希尔体系有大变动。

我认为另一个假设也不会成立。巴菲特卸任后，伯克希尔不会停止收购优质企业。考虑到伯克希尔当今庞大的规模，以及积极投资时代的来临，我认为理想的收购机会必将出现，伯克希尔600亿美元的现金会找到好去处。

最后，我想谈谈伯克希尔过去五十载的辉煌成就是否具有借鉴意义。

答案显然是肯定的。在巴菲特任期的早些年里，伯克希尔面临一个艰巨的任务，就是让小公司壮大成为大型杰出企业。伯克希尔有一位深思熟虑的领导，他在不断提升自身能力的同时，广纳英才。通过避免官僚制度，并长期依靠这位英明领导，伯克希尔完成了任务。

反观一家典型的大企业的情况：非常官僚化的总部，走马灯式的CEO制度。通常CEO在59岁左右上任，还没等静下心来思考问题，就到了强制退休年龄，不得不卷铺盖走人。

我相信伯克希尔体系应该以不同的形式加以大力推广，而官僚制度最败坏之处应被视为毒瘤，坚决铲除。一个最好的案例是乔治·马歇尔上将对美军官僚体制的改革。二战期间，他获得国会的授权，在选用将领时，不受军龄限制。这一打破官僚主义的做法，让他帮助美国打赢了二战。

此致，  
查理·芒格
